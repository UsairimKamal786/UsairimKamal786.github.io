# Usairim Web Tech Games

Welcome to **Usairim Web Tech Games**, a stylish interactive web page designed to showcase fun links to Games and Agents.

## Features

- **Neon Gaming Style:** Dark background with glowing neon buttons for a modern, futuristic vibe.  
- **Interactive Buttons:** Clickable buttons for navigating to:
  - `Games` → games.html  
  - `Agents` → agents.html  
- **Responsive Design:** Works on desktop and mobile devices.  
- **Copyright:** All rights reserved by Usairim Kamal.

## How to Use

1. Clone or download the project folder.  
2. Open `index.html` in your web browser.  
3. Click the buttons to explore Games or Agents pages.  
4. Customize links or styles in `index.html` as needed.


## Customization

- Change button links in `<a href="...">` to navigate to your own pages.  
- Adjust colors, fonts, and styles in the `<style>` section of `index.html`.
